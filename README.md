# ClassInsight - 스마트 질의응답 시스템

## 🎯 시스템 개요

ClassInsight는 여러분의 질문에 대해 가장 정확하고 유용한 답변을 제공하는 똑똑한 AI 시스템입니다. 
마치 도서관 사서와 인터넷 검색 전문가, 그리고 연구 논문 전문가가 모두 함께 여러분을 도와주는 것과 같습니다.

## 🤖 무엇을 할 수 있나요?

### 1. 똑똑한 검색
- **내부 자료 검색**: 시스템에 저장된 문서에서 관련 정보를 찾습니다
- **인터넷 검색**: Google을 통해 최신 정보나 일반 지식을 검색합니다
- **학술 논문 검색**: arXiv에서 최신 연구 논문을 찾습니다

### 2. 연속 대화 (Multi-turn)
- 이전에 나눈 대화를 기억하고 맥락을 이해합니다
- "그것"이나 "앞에서 말한 것"과 같은 표현도 이해할 수 있습니다
- 자연스러운 대화처럼 질문을 이어갈 수 있습니다

### 3. 종합적인 답변
- 여러 출처의 정보를 종합하여 완전한 답변을 제공합니다
- 정보의 출처를 명확히 표시합니다
- 확실하지 않은 정보는 추측하지 않습니다

## 🔍 어떻게 작동하나요?

### 단계별 검색 과정

```
질문 입력 → 쿼리 최적화 → 내부 검색 → 결과 평가
                                      ↓
                                   충분함? 
                                   ↙    ↘
                             답변 생성    외부 검색
                                        ↓
                                   Google/arXiv 검색
                                        ↓
                                     답변 생성
```

#### 1️⃣ 질문 분석
- 사용자의 질문을 분석하여 검색에 최적화된 키워드를 생성합니다
- 이전 대화 내용을 고려하여 맥락을 파악합니다

#### 2️⃣ 내부 검색 우선
- 먼저 시스템에 저장된 문서에서 관련 정보를 찾습니다
- 검색 결과가 충분하다면 바로 답변을 생성합니다

#### 3️⃣ 외부 검색 (필요시)
- 내부 검색 결과가 부족하면 외부 검색을 실행합니다
- 질문의 성격에 따라 Google 또는 arXiv를 선택합니다
  - **Google**: 최신 뉴스, 일반 정보, 실시간 데이터
  - **arXiv**: 학술 연구, 논문, 기술적 내용

#### 4️⃣ 통합 답변 생성
- 모든 검색 결과를 종합하여 완전한 답변을 만듭니다
- 정보의 출처를 구분하여 표시합니다

## 💬 사용법

### 웹 인터페이스
1. 브라우저에서 시스템에 접속합니다
2. 하단의 입력창에 질문을 타이핑합니다
3. "전송" 버튼을 클릭하거나 Enter 키를 누릅니다
4. 시스템이 답변과 함께 검색 과정 정보를 보여줍니다

### 새 대화 시작
- "새 대화" 버튼을 클릭하면 새로운 대화 세션이 시작됩니다
- 각 세션은 독립적으로 대화 맥락을 관리합니다

### 시스템 정보 확인
- 화면 상단의 상태 표시등으로 시스템 준비 상태를 확인할 수 있습니다


## 🔧 시스템 구성

### 기술 스택
- **백엔드**: Python FastAPI - 질문 처리와 검색 로직
- **프론트엔드**: HTML/CSS/JavaScript - 사용자 인터페이스
- **검색 엔진**: Elasticsearch - 내부 문서 검색
- **AI 모델**: OpenAI GPT - 질문 이해와 답변 생성
- **외부 API**: Google Search, arXiv - 외부 정보 검색

### 컨테이너 기반 운영
- Docker를 사용하여 안정적인 서비스 제공
- 백엔드와 프론트엔드가 독립적으로 운영
- 자동 상태 확인과 복구 기능

## 🌟 특장점

### 1. 다층 검색 시스템
- 단순히 하나의 검색 방법에 의존하지 않습니다
- 내부 → 외부 순서로 체계적인 검색을 수행합니다
- 질문에 가장 적합한 검색 방법을 자동으로 선택합니다

### 2. 맥락 이해 능력
- 이전 대화를 기억하여 연속적인 질문에 답할 수 있습니다
- "그것", "위에서 말한" 등의 지시어를 이해합니다
- 대화의 흐름을 고려한 자연스러운 응답을 제공합니다

### 3. 투명한 정보 제공
- 어디서 정보를 가져왔는지 명확히 표시합니다
- 검색 과정을 사용자가 확인할 수 있습니다
- 불확실한 정보는 추측하지 않고 솔직하게 알려줍니다

### 4. 실시간 정보 접근
- 내부 문서뿐만 아니라 인터넷의 최신 정보까지 활용합니다
- 학술 논문을 통해 전문적이고 신뢰할 수 있는 정보를 제공합니다
- 다양한 출처의 정보를 종합하여 균형 잡힌 답변을 만듭니다

## 📞 시작하기

1. **시스템 접속**: 웹브라우저를 열고 제공된 주소로 접속합니다
2. **상태 확인**: 화면 상단의 상태 표시등이 초록색인지 확인합니다
3. **질문하기**: 하단 입력창에 궁금한 것을 자유롭게 입력합니다
4. **답변 확인**: 시스템이 제공하는 답변과 검색 정보를 확인합니다
5. **대화 계속**: 추가 질문이나 관련 질문을 자유롭게 이어갑니다
